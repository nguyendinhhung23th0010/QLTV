# QLTV - Quản Lý Thư Viện

Đây là dự án PHP thuần để quản lý thư viện, bao gồm các chức năng:
- Thêm/sửa/xóa sách
- Quản lý mượn trả
- Quản lý người dùng
- Thống kê cơ bản

## Công nghệ sử dụng
- PHP
- MySQL
- Bootstrap
- XAMPP
